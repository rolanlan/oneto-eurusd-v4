④ BUG-09 评分公式

AI Committee 顶部 N%（一致性 / Alignment） 来源文件: src/types/Vote.js → aggregateVotes() 公式: sell_weight = Σ agent.weight × (0.5 if RISK else 1.0) for agents voting SELL buy_weight = Σ agent.weight × (0.5 if RISK else 1.0) for agents voting BUY confidence = round( max(sell_weight, buy_weight) × 100 ) 示例（4代理投SELL，权重0.30+0.20+0.20+0.15，RISK折半0.15×0.5=0.075）: sell_weight = 0.30+0.20+0.20+0.075 = 0.775 → confidence = 78% 含义: 占优方向的加权投票占比。85% = 委员会85%权重偏向同一方向。 Decision Engine N（方向评分 / Direction Score） 来源文件: src/core/SignalEngine.js → _decide() 公式: wSum = 1 − risk_weight dirScore = round( Σ (agent.score × agent.weight / wSum) ) 排除Risk代理 含义: 0=纯做多，50=中性，100=纯做空 触发: >55 → SELL；<45 → BUY；45–55 → NEUTRAL 72 含义: 方向偏空，接近标准阈值55，未达强烈卖出水平。 Confidence %（置信度） 公式: baseConf = |dirScore − 50| × 2 riskPenalty = max(0, (riskScore − 55) × 0.3) finalConf = clamp(0,100, baseConf − riskPenalty + MTFadj) dirScore=72 → baseConf = |72−50|×2 = 44% 低于 min_confidence 65% → 触发 LOW_CONFIDENCE → NO_TRADE 门控
